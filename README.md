# leisu-website-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能来自不同时期、不同主题或不同场景，仓库本身不针对任何单个域名或具体网站。所有页面均以原始文件形式保存，方便查阅、分享与历史回顾。

## 目录说明

```
leisu-website-archive/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的资源文件（如图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 页面作为一个独立文件存放，文件名通常反映内容或归档时间。
- `assets/`：若页面依赖外部资源，则统一放置于此目录下，按需组织子文件夹。

## 页面归档说明

- 所有页面均为静态 HTML 文件，可直接在浏览器中打开查看。
- 归档的页面可能包含不同时间点、不同主题或不同来源的内容，请以实际页面信息为准。
- 部分页面可能包含过时、不完整或仅供特定场景使用的信息，请结合上下文理解。

## 维护说明

- 本仓库不定期更新，新增的页面会按时间或主题顺序添加至 `pages/` 目录。
- 如发现页面存在显示异常或链接失效，欢迎提交 Issue 或 Pull Request。
- 仓库维护者会尽力保证归档文件的完整性和可访问性，但不对页面内容的准确性或时效性做出承诺。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 通过浏览器打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 如需引用页面资源，请确保 `assets/` 目录与页面文件保持相对路径一致。
